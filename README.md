# 📱 CodeQuest – Quiz Application

**CodeQuest** is an interactive quiz application designed to make learning fun, competitive, and engaging. Users can test their knowledge by attempting quizzes, competing on leaderboards, and progressing through level-based challenges.

---

## 🚀 Features

### 🧠 Attempt Quizzes

* Users can play quizzes from various categories.
* Each quiz is designed to test knowledge, accuracy, and speed.

### 🏆 Leaderboard System

* Scores are updated in real time after each quiz.
* Users can compete with others and aim for the top position on the leaderboard.

### 🎮 Level-Based Progression

* The app includes a level-by-level quiz system.
* Difficulty increases as users progress to higher levels.

### ✍️ Create Your Own Quiz

* Users can create custom quizzes.
* They can add their own questions, options, and correct answers.

### 🔗 Join Quiz via Code

* Each created quiz generates a unique join code.
* Users can share this code with friends so they can join and attempt the quiz.

### 🔥 Daily Streak Quiz

* A daily streak system encourages consistent participation.
* Users can maintain streaks by attempting quizzes regularly.
